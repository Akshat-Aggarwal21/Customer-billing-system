## Description

This is a simple Customer Billing System developed in C++.
The program allows users to store and manage customer billing records using file handling.

Customer details such as name, account number, address, mobile number, payment, and balance are stored in a binary file. The system can also search and display customer records.

## Features

* Add new customer accounts

* Store customer data in a file

* Search customer by customer number

* Search customer by name

* Display billing details

* Calculate new balance after payment


## Technologies Used

* C++

* File Handling (fstream)

* Structures

* Binary Files
